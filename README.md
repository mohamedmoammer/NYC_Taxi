# 🚖 NYC Taxi Trip Analysis | Power BI

## 📌 Project Overview

This project analyzes the New York City Taxi dataset using Power BI to uncover operational trends, trip patterns, customer behavior, and revenue insights. The dashboard provides interactive visualizations that help identify peak demand periods, optimize taxi operations, and support data-driven business decisions.

---

# 🎯 Business Objectives

- Analyze taxi trip demand across NYC.
- Monitor trip volume over time.
- Identify peak pickup and drop-off locations.
- Evaluate fare and revenue trends.
- Analyze trip distance and duration.
- Discover customer travel patterns.
- Support operational and business planning.
